# health_Care

Dynamic AI-driven telemedicine and healthcare management platform built with Next.js, TypeScript, and PostgreSQL.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Main Capabilities

- Role-based dashboards (patient, doctor, medicalist, superadmin)
- Appointment and consultation workflows
- Prescription and transaction management
- AI-assisted doctor recommendation and disease hotspot analysis
- Nearby medical store map integration
- Full project documentation and dissertation report
